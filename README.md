# River Raid study case for Women in Tech: Carol Shaw

- Small River Raid app created to better visualize the importance of Carol's code and contributions years ago.

- Blog post link: [Link to Blog Post]

## Tools:

- React
- TypeScript
- Vite
- pnpm

## How to run it:
1. Install dependencies with:
 `pnpm install`
2. Start the development server with: 
 `pnpm run dev`
3. Open the app in your browser at: 
 `http://localhost:5173`
